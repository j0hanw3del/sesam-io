# Welcome to the Sesam User Directory App! 🚀🎉

This is a simple and intuitive React application that allows you to explore and filter through a list of users, their contact information, addresses, and companies.

## Prerequisites 📚

Before you get started, make sure you have the following installed:

- [Node.js](https://nodejs.org/en/download/) (LTS version recommended)
- [npm](https://www.npmjs.com/get-npm) (usually comes with Node.js)

## Getting Started 🏃‍♀️🏃‍♂️

Follow these simple steps to start the app locally:

1. Clone the repository:

```bash
git clone https://github.com/j0hanw3del/sesam-io.git
cd sesam-io
```

2. Install the required dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

That's it! The app should now be running on [http://localhost:3000](http://localhost:3000) 🌟

Open up your favorite browser and start exploring the User Directory App! 😃

## Features ✨

- Search users by name 🔍
- Sort users by name, company, or city 📊
- View user details on a separate page 📖
